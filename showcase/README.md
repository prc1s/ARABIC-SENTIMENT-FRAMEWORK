# Research Showcase Site

Static website for presenting the Arabic sentiment analysis thesis and its saved experiment artifacts.

## Preview locally

From the repository root:

```bash
python -m http.server 8000
```

Then open:

```text
http://localhost:8000/showcase/
```

## Files

- `showcase/index.html`: page structure
- `showcase/styles.css`: visual design and responsive layout
- `showcase/data.js`: thesis summary, dataset facts, and metrics copied from saved artifacts
- `showcase/app.js`: lightweight rendering logic for the interactive results section
- `showcase/assets/arabic_sentiment_fyp_Final.pdf`: downloadable thesis PDF
